# v0.04.1 发布记录

## 新增内容

- 新增桌面端“依赖”页，集中显示 `yt-dlp`、`ffmpeg`、抖音下载器、Whisper、FunASR、Microsoft Edge、DeepSeek API Key 的状态。
- 支持一键下载并配置 `yt-dlp` 和 `ffmpeg` 到应用工作区，不再要求普通用户手动找路径。
- 新增 CLI：
  - `dependencies report`
  - `dependencies install`
- 健康检查横幅增加“去依赖”入口。
- 更新安装说明和使用说明，明确 Docker、Python、Node.js、Rust、VS Code 都不是普通用户必需项。

- DeepSeek API Key 和平台账号登录态仍需要用户自己填写或登录。
