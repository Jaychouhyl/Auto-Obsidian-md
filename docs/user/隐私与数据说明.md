# 隐私与数据说明

## 本地优先

Knowledge Studio 个人版按本地桌面软件设计。配置、队列、缓存、日志、账号资料和输出文件默认保存在你的电脑上。

## 不做的事情

- 不提供云同步。
- 不托管你的平台账号。
- 不代保存平台密码。
- 不把诊断包自动上传到服务器。
- 不把本地文件自动公开发布。

## 可能发送到外部的内容

如果你启用了 LLM 摘要，正文、转写文本或提取出的网页/PDF 内容会发送到你配置的模型 API，例如 DeepSeek 或其它 OpenAI-compatible 服务。是否发送、发送到哪里，取决于你在“配置”页填写的 Base URL 和 API Key。

## 本机敏感数据

这些内容不要公开分享：

- `config.toml`
- `douyin-config.yml`
- `accounts/`
- `.cookies.json`
- `data/`
- `cache/`
- 备份 zip
- 诊断 zip

诊断包会脱敏常见 Key/Token/Cookie 字段，并排除账号浏览器缓存，但仍建议分享前自行检查。

## 删除数据

如果你想清理本机数据，可以先在软件里创建备份，再删除本机工作目录里的 `cache`、`logs`、`data` 或账号资料。删除账号资料会导致平台需要重新登录。
