# Knowledge Studio 个人版快速开始

## 适用范围

这份说明面向本地自用版。软件只在你的 Windows 电脑上运行，内容默认写到本地目录、Obsidian vault、HTML、CSV 或 Notion。它不提供云同步、不托管账号、不代保存平台密码。

## 第一次打开

1. 安装并启动 `Knowledge Studio 个人版`。
2. 在“配置”页选择输出目录。用 Obsidian 就选你的 vault；不用 Obsidian 就选一个普通文件夹。
3. 需要 AI 摘要时，打开“启用 LLM”，填写 DeepSeek 或其它 OpenAI-compatible API Key。
4. 打开“依赖”页，点击“安装缺失工具”。视频下载通常需要 `yt-dlp` 和 `ffmpeg`。
5. 打开“账号”页，按需登录抖音、B 站、YouTube 等账号。密码只在平台网页里输入，软件不保存密码。
6. 回到“运行”或“导入”页，导入链接、RSS、本地目录、网页或抖音收藏。

## 推荐日常流程

1. 打开软件。
2. 先点“诊断与日志”里的“重新检查”。
3. 在“账号”页确认要用的平台账号仍然有效。
4. 在“导入”页加入来源。
5. 在“运行”页处理队列。
6. 到输出目录检查生成的 Markdown、HTML 或 CSV。

## 出问题时

1. 先看“诊断与日志”页的处理建议。
2. 常见问题先处理：API Key、账号登录态、输出目录权限、`ffmpeg`、`yt-dlp`。
3. 仍然失败时，点击“导出诊断包”。诊断包在本机生成，会脱敏常见 Key/Token，但分享前仍建议自己检查。
