# 平台能力边界

## 核心原则

软件帮助用户整理自己可访问的学习资料，不保证任何第三方平台接口永久稳定。

## 抖音、B 站、YouTube、TikTok

这些来源依赖平台页面、登录态、Cookie、下载器和平台规则。以下情况都可能导致抓取数量变少或失败：

- 登录态过期。
- 平台要求验证码。
- 平台触发风控或限流。
- 视频不可见、已删除或区域不可访问。
- 下载器暂未适配平台页面变化。
- 用户输入的收藏数量超过平台当前可返回数量。

## 商业文案建议

可以写：

> 支持在用户已登录状态下导入公开视频、收藏、播放列表、网页、RSS、本地文件，并沉淀为 Markdown/HTML/CSV/Notion。

不要写：

> 永久稳定抓取所有抖音收藏。

> 绕过平台限制下载任何视频。

> 无需账号即可抓取私人收藏。

## 推荐交付口径

这是一个本地知识整理工具，不是平台内容破解工具。用户必须对自己导入的内容拥有访问权限，并遵守对应平台规则。
