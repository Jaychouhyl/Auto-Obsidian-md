# Knowledge Studio 常见问题

## 必须使用 Obsidian 吗？

不必须。你可以只输出 Markdown 文件，也可以同时输出 HTML、CSV。Obsidian 只是最顺手的知识库读取方式。

## 为什么视频处理失败？

常见原因是 `ffmpeg`、`yt-dlp`、转写工具缺失，或者平台登录态失效。先到“依赖”页重新检测，再到“账号”页校验账号。

## 为什么抖音收藏抓取数量不等于我输入的数量？

平台返回结果会受到登录态、收藏可见性、平台规则、风控、分页和去重影响。软件只能处理当前账号在当前状态下能访问到的内容，不承诺每次都完整拉取所有收藏。

## API Key 放在哪里？

在“配置”页填写。保存后界面不会回显 Key。备份和诊断包会对常见 Key/Token 字段做脱敏，但你仍然不要公开分享自己的配置文件。

## 账号密码会被保存吗？

不会。账号登录使用系统 Edge 打开平台网页，密码只在平台网页登录。软件保存的是本机浏览器资料目录和必要的账号元数据，用于下次复用登录态。

## 可以商用吗？

可以作为本地桌面工具交付给用户自用。建议不要承诺平台抓取永久稳定，也不要代用户托管账号、Key 或数据。公开分发前建议补代码签名和用户协议。

## 为什么 Windows 提示未知发布者？

当前安装包没有代码签名证书。个人使用可以继续安装；正式对外销售时建议购买 Windows 代码签名证书。

## 诊断包会上传到哪里？

不会上传。诊断包只保存在本机 `logs` 目录，用户自己决定是否发送给维护者。
